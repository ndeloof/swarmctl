# docker swarmctl version

<!---MARKER_GEN_START-->
Display Swarm Control version

### Options

| Name             | Type     | Default | Description                                                    |
|:-----------------|:---------|:--------|:---------------------------------------------------------------|
| `-f`, `--format` | `string` |         | Format the output. Values: [pretty \| json]. (Default: pretty) |
| `--short`        |          |         | Shows only Swarm Controller's version number.                  |
